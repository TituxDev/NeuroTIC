Detailed Description

Pointer registry and deferred cleanup implementation.

Author
Oscar Sotomayor (TituxDev).

Implements a simple global pointer registry used during development to track dynamically allocated memory blocks and release them collectively.

The registry provides a coarse-grained cleanup mechanism intended for small experiments and early framework development.

This module is auxiliary in nature and may be replaced by a more capable memory subsystem in the future.
