Detailed Description

Implementation of memory management functions.

Provides functions for tracking dynamically allocated memory blocks and ensuring they are freed at program termination.
Uses a dynamic registry to store pointers to allocated memory, and registers a cleanup function with atexit() to automatically free all tracked memory when the program exits.

◆ cleanmemory()

void cleanmemory ( void  )

Frees every tracked memory block, for every owner, and empties the tracking registry.

Frees every memory block registered under every tracked owner, then frees each owner's register and the mem_tracker array itself, resetting mem_track to zero.

Warning
Not meant to be called manually – it is registered with atexit() (by createowner(), the first time it runs) so that all tracked memory is freed automatically when the program exits. Calling it while the tracking system is still in use may cause double frees or dangling pointers.
Attention
Assumes every registered block was allocated with malloc() or a compatible allocator (e.g. calloc()), and can be safely freed.
See also
deleteowner() to remove a single owner instead of everything.

Creates a memory register entry for a specific memory owner.

Return values
NULL
  • mem is NULL.
  • owner is non-NULL but the internal owner or register table could not be grown (allocation failure).
Note
If owner is NULL and mem is non-NULL, mem itself is returned unchanged – but it is neither registered nor freed in that case; the caller remains responsible for it.

Ensures owner is tracked (creating it via createowner() if needed), then registers mem under that owner if it is not already present, growing the owner's block registry by one slot. If mem was already registered under this owner, it is returned unchanged without growing the registry.

References createowner(), and mem_format::mem_register.

◆ deleteowner()

unsigned char deleteowner ( void *  owner)

Deletes a memory owner and all associated memory blocks.

Return values
1owner is NULL.
2the compacted tracker array could not be allocated (only reachable when 2 or more owners are tracked).

Deletes owner and every memory block registered under it.

When two or more owners are tracked, searches for owner and, if found, frees its registered blocks and compacts the remaining owners into a freshly allocated array.

When at most one owner is tracked, deleting the sole remaining owner (if owner matches it) is equivalent to clearing the entire tracking system, so cleanmemory() is called directly instead of shrinking an array down to zero elements.

If owner is not currently tracked, nothing is deleted.

References cleanmemory(), mem_format::mem_register, and mem_track.
