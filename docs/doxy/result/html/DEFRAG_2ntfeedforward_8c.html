Detailed Description

Implementation of predefined network topologies.

Provides functions to generate common wiring configurations for NeuroTIC networks, initializing `bff_wiring` and neuron `bff_idx` values.

This module is intended for convenience; users may also define their own custom `bff_wiring` structures in the main code.
