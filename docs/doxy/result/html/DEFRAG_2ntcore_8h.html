Detailed Description

Core data structures for the NeuroTIC framework.

This header defines the fundamental structures used to assemble and operate neural networks in NeuroTIC. These types describe neurons, inter-layer wiring, buffer systems, and the network container itself.

All higher-level behaviors—construction, initialization, activation dispatch, training, and persistence—are built on top of these definitions. The core structures remain intentionally small, explicit, and fully transparent, making every network inspectable at runtime and fully compatible with both automated and hand-written topologies.
