Detailed Description

This structure is used to associate memory blocks with their respective owners (e.g., modules or components within the NeuroTIC system). Each mem_format instance contains a pointer to the owner, a dynamic array of pointers to registered memory blocks, and a counter for the number of registered blocks.
The mem_tracker is a dynamic array of mem_format structures that allows for efficient tracking and management of memory ownership and associated memory blocks throughout the lifecycle of the program.
