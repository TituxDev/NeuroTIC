https://tituxdev.github.io/NeuroTIC/src/CPU/ntcalculate.c

Provides weighted sum, activation, and full forward propagation for NeuroTIC networks. Assumes sequential layer-by-layer calculation.

◆ feedforward()

data_t * feedforward ( net_s *  net)

Executes full feedforward propagation.

All layers are evaluated sequentially. Each neuron is activated in order of declaration.

Parameters
netPointer to the network.
Returns
Pointer to the output buffer of the final layer.

Iterates layer-by-layer to maintain deterministic ordering. No buffering is required since outputs of previous layers are already wired via pointer connections.

References activate(), net_s::layers, net_s::neurons, net_s::nn, and net_s::out.

Referenced by backpropagation().
