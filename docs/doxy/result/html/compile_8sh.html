Detailed Description

Compilation helper for NeuroTIC projects.

This script resolves project dependencies for a given user project, compiles NeuroTIC and user source files, generates static libraries, and links the final executable.

Usage: ./compile.sh <project_location> <project_name> <platform>

Example: ./compile.sh ./MyProject main x86_64
