Detailed Description

Auxiliary memory tracking utilities.

Author
Oscar Sotomayor (TituxDev).

Declares a minimal pointer tracking interface used to register dynamically allocated memory blocks for deferred collective release.

This interface is primarily intended for internal use during development and experimentation.

The module does not define ownership semantics and may be replaced by a more complete solution as the framework evolves.
