# Contributing to NeuroTIC

Thank you for your interest in contributing to NeuroTIC!

## Contribution License Agreement

By contributing to NeuroTIC, you agree that:
- Your contributions will be licensed under both:
  - GNU GPLv3 for open source use
  - Commercial license for proprietary use
- You retain copyright of your contributions
- The project maintainer may license your contributions under commercial terms

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Submit a pull request

## Code Standards

- Follow the existing code style
- Write clear commit messages
- Update documentation as needed
- Ensure compatibility with embedded systems
